# 补充知识：如何运行代码（计算机组成原理）
1. 将代码数据读入到内存中，分配PCB，形成了进程。在该进程中,代码变成了一条一条指令。

2. 
```c++
    while(code != EOF) {
        // 第一步
        将PC指向的指令读入IR中
        // 第二步
        令 PC 指向下一条待执行的指令（一般就是: PC += 当前指令长度）
        // 第三步
        控制器解析执行指令
    }
```

# 进程切换
## 是什么？
当前进程时间片到后，CPU会切走当前进程，切入其他进程。

## 为什么要切换？
1. 分时操作系统是基于时间片调度轮转的操作系统，追求调度公平，所以进程才有了时间片，然而时间片一到就得换下一个进程。
2. 方法：并发

## 怎么切换？
1. CPU的寄存器只有一套，所以当前进程切走后，要将寄存器数据存入PCB中（古老版本），从而让寄存器空闲出来，便于下一个进程的使用。
2. 当该进程被再次切入时，将PCB中的寄存器数据写入寄存器中，就能接着之前的进度，继续执行任务。
3. 而这套寄存器数据就叫做**上下文数据**。
