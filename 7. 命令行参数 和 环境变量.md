# 命令行参数

## 概念
1. 在 Terminal 输入命令
2. shell 会拿到该命令（字符串的形式存储命令）
3. 按照空格隔开，形成多个参数

## c 的主函数获取命令行参数

```c
    int main(int argc, char *argv[]) {}
```

1. argc: 参数个数
2. argv: 参数列表

## 举例
```
    ls -a -l
```

1. argc：3
2. argv: [ls, -a, -l]

# 环境变量

## 概念
1. 形式：key->value
2. 本质：具有全局属性的变量
3. 位置：在系统配置文件中（硬盘）
4. 特性：每一个进程都有一个环境变量副本，而且可以被子进程继承

## 获取环境变量

### 代码中

```c
    int main(int argc, char *argv[]， char *env[]) // 类似于命令行参数传参的方式  
    {
        // 使用该函数，获取特定的环境变量
        getenv("PWD");
    }
```

### terminal中
```c
    env //获取所有的环境变量
    echo $PWD // 获取当前工作路径
    echo $HOME // 用户主目录
    ...... 
```

## 环境变量应用

1. 将命令所在目录配置到 PATH 中，可以直接运行该命令
2. 进程间传递数据
3. 作为配置成功的表现（如 g++ -v）
